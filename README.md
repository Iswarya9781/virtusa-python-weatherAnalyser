📌 Project Overview

This project is a Python-based web application that fetches real-time weather data, analyzes historical trends, and predicts future temperature. 
It helps users understand weather patterns instead of just viewing raw data.

🎯 Objectives
Fetch real-time weather data using API
Store historical data for analysis
Visualize temperature trends
Predict future temperature using regression
Display results in an interactive dashboard

⚙️ Technologies Used
Python
Flask (Web Framework)
Pandas (Data Analysis)
Matplotlib (Data Visualization)
Scikit-learn (Prediction Model)
OpenWeatherMap API (Weather Data)

🚀 Features
🌍 Search weather by city
📊 Displays average temperature & humidity
📈 Shows temperature trend graph
🔮 Predicts next day temperature
💾 Stores historical data in CSV
